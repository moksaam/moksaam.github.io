DOM Manipulation

Here we will be showing different ways to change the DOM using Javascript.

The Create Element button will create a new DIV element and append it to our current DIV.

The Append Child button will create a new P element and append it to our newly created DIV. It will also create a "new" DIV element and append that to the body of the document.

The Insert Before will create a large DIV and place it at the top of this div.

The Remove Child button will remove the divs from inside our main div.
